天线计算工具

用于天线、OTA、匹配和射频链路分析的在线计算。

Calculator Workspace

实用射频计算器

Module 01

频率与波长计算

输入频率后自动换算全波长、半波长和四分之一波长。

Module 02

Band 频段查询

支持 LTE / 5G NR，输入如 38、b38、n78、Band 41。

Module 03

阻抗匹配网络综合工具

优先评估 2 元件(L型)匹配方案以降低寄生效应风险。若全频段驻波不及格，则自动切换至带有寄生参数约束（C > 0.3pF, L > 0.3nH）的 3 元件(Π 型)综合方案，并输出可用的标准物料组合。

1. 设定测试频点与阻抗 (首中尾)

等待执行宽频综合...

2. 推荐标准元器件组合

Module 05

自由空间路径损耗

按米和 MHz 计算 FSPL。

Module 06

效率与 dB 换算

按功率效率换算，50% 对应约 -3.01 dB。

Module 07

dBm 与 Watt 转换

双向换算功率单位，网页端按当前输入侧实时更新另一侧。

Module 08

多天线合成增益

输入多个天线增益，自动输出非相干和相干合成结果。

Module 09

有源与无源指标等效预估

按你当前桌面版逻辑，输入增益和效率，反推方向性，并联动计算 EIRP / TRP / EIS / TIS。

[无源] 增益与方向性

[有源发射 TX]

[有源接收 RX]

Module 10

VSWR 与 Return Loss 换算

同时给出反射系数、失配损耗和传输能量。

Module 11

远场测试距离评估

根据天线物理尺寸和频率计算 Fraunhofer Distance。

Module 12

ZOR 零阶谐振器参数综合

输入目标频率以及微带主体的本征参数，反推所需超材料(CRLH)部分的寄生电感和寄生电容。并顺带提供通过物理尺寸预估接地过孔/短截线引脚电感的快捷工具。

[输入] 设计目标与宿主

[综合] 所需 CRLH 等效参数

[辅助] 接地过孔/短截线电感预估

Module 13

微带线特征阻抗综合

基于 Wheeler/Hammerstad 经验公式，实时预估微带线阻抗（输入 W 出 Z0）或自动用二分法倒推所需馈线线宽（输入目标 Z0 出 W）。

[基材] 板材分布参数

[预估] 已知线宽 -> 阻抗

[综合] 目标阻抗 -> 线宽

Module 14

品质因数 (Q) / 分数带宽换算

核算小型化天线学术指标，逼近物理极限。
